# quick_crop_image_tool
## 快速裁剪图片工具
- cut_images_tool:  是图片快速裁剪小工具。
  - 	可设置需要裁剪图片的路径，裁剪后图片的存放路径，保存的文件名；
  - 	读取图片/图片文件夹后，需要手动点击下一张；
  - 	图片可以通过鼠标拖动、缩放，左键单击选择裁剪区域，并保存拖动和缩放时区域不变；
  - 	在框选一个图片区域后，可自动保存，文件名不重复。
